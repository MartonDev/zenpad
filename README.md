# ZenPad

As _zen_ as it gets

[ZenPad](http://zenpad.now.sh) is a minimal web app for taking quick notes.

## Features:

- Ligatures using [JetBrains Mono](https://www.jetbrains.com/lp/mono/) font
- Persistence using localStorage
- Spellcheck using your browser
